Glosario Web3

[ Base de Datos Descentralizada de Conocimiento ]

Blockchain

Tecnología de registro distribuido que permite la transferencia de datos de manera segura y transparente sin la necesidad de intermediarios.

Contratos Inteligentes

Programas informáticos que utilizan firmas digitales para validar la ejecución automática de acuerdos una vez que se cumplen ciertas condiciones preestablecidas.

Criptografía de Clave Pública

Sistema de seguridad asimétrico que usa dos claves: una pública para recibir datos, y una privada para firmar y descifrar la información.

Descentralización

Distribución de funciones y autoridad lejos de un grupo único de control. Al distribuir el libro mayor entre múltiples nodos, se elimina el riesgo de un único punto de fallo.

Firma Digital

Secuencia cifrada generada mediante la clave privada del usuario. Verifica la identidad del emisor y la integridad del mensaje o transacción.

Hash

Función criptográfica que convierte una entrada de datos en una cadena de caracteres de longitud fija. Cualquier cambio en los datos originales resulta en un hash completamente diferente, asegurando la integridad.

Libro Mayor Distribuido

Base de datos consensuada y replicada en múltiples nodos de una red. Se mantiene sincronizada de forma inmutable y transparente para todos los participantes.

Mecanismos de Consenso (PoW y PoS)

Procesos para acordar el estado de la cadena. PoW (Prueba de Trabajo): Los mineros compiten resolviendo problemas criptográficos. PoS (Prueba de Participación): Los validadores son seleccionados según la cantidad de criptomonedas que apuestan.

Minería de Datos

Proceso que implica la resolución de complejos problemas criptográficos para validar transacciones y agregar nuevos bloques a la cadena a cambio de recompensas.

Nodos

Dispositivos que participan en la red y mantienen una copia del libro mayor. Pueden ser nodos completos, ligeros, de minería o de validación.

Nonce

Un número utilizado una sola vez. Los mineros lo ajustan continuamente durante el proceso de minería hasta producir un hash que valide el bloque actual.

Raíz de Merkle (Merkle Root)

Un hash maestro que representa y resume todas las transacciones incluidas en un bloque. Se utiliza para verificar rápidamente la integridad de los datos sin revisar cada transacción individualmente.

Red P2P (Peer-to-Peer)

Red donde cada nodo actúa como un "peer" (igual) que se comunica directamente con otros nodos, compartiendo datos y recursos de forma descentralizada y sin servidores centrales.
